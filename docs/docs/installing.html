Installing Open Darkroom

MacOS

  1. Download the latest version of Open Darkoom from the Downloads Section
  2. Unzip, and drag to the Applications folder
  3. Right click the Open Darkoom icon and select open, instead of double clicking.
  4. You will likely see a message saying it is from an unidentified developer, click ok. If you do not see an ok button make sure you opened the app by right clicking it as stated above.

Windows

  1. Download the latest version of Open Darkoom from the Downloads Section
  2. Run Darkroom-x.x.x.Setup.exe and wait for it to install.
  3. If you get a message saying "Windows Protected your PC" select "More Info" and "Run Anyways"
